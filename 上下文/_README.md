# 实验上下文目录

此目录用于存放实验相关的背景知识文件，后端启动时自动加载合并为用户上下文。

## 支持格式

| 格式 | 说明 |
|------|------|
| `.md` | Markdown（推荐，公式用 `$...$` / `$$...$$`） |
| `.txt` | 纯文本 |
| `.pdf` | PDF 文档（仅文字版，扫描件无效） |
| `.docx` | Word 文档 |

## 添加上下文

直接将文件放入此目录，重启后端或调用 `POST /api/experiment/context/reload` 即可加载。

## 文件用途

| 文件 | 加载位置 |
|------|---------|
| 所有匹配文件 | AI 助手的 System Prompt（对话背景知识） |

## 示例

### 公式推导.md

```markdown
# 公式推导

## 驱动扭矩

$$T_z \approx \sigma d B^2 R^4 (\Omega_M - \Omega_D)$$

其中 $\Omega_M$ 为驱动磁铁转速，$\Omega_D$ 为探针转速。

## 粘性阻力矩

$$T_v \approx -\frac{\eta R^4 \Omega_D}{h}$$
```

### 实验步骤.md

```markdown
# 实验步骤

1. 准备样品，注入样品池
2. 启动磁场，实现稳定悬浮
3. 启动旋转驱动，记录视频
4. 使用本平台进行转速跟踪与分析
```

## 注意事项

- 公式用 LaTeX 语法（`$...$` 行内，`$$...$$` 块级）
- PDF 必须是文字型 PDF（扫描件/图片型无法提取文字）
- 文件按名称排序后合并，用 `---` 分隔
- 文件编码建议 UTF-8
