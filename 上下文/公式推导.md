\# 驱动扭矩 $T\_z$ 的详细推导（对应原文公式 (4)）



\## 1. 物理问题描述



在论文装置中，\*\*驱动磁铁\*\*（四块方形磁铁）以角速度 $\\Omega\_M$ 旋转，产生旋转磁场。该磁场穿过紧贴石墨盘的铝盘（厚度 $d$，半径 $R$，电导率 $\\sigma$），在铝盘中感应出涡流。载流铝盘在磁场中受到洛伦兹力，形成驱动扭矩 $T\_z$，试图使铝盘跟随磁场旋转。最终我们得到近似公式：

$T\_z \\approx \\sigma d B^2 R^4 (\\Omega\_M - \\Omega\_D)$，其中 $\\Omega\_D$ 是铝盘实际旋转角速度，$B$ 是磁场特征幅值。



\## 2. 基本假设



为简化推导，作以下合理假设（与论文一致）：

\- 铝盘极薄：$d \\ll R$，忽略厚度方向电流变化。

\- 磁场径向变化缓慢：在铝盘半径范围内 $B(r) \\approx B\_0$ 为常数。

\- 忽略铝盘涡流对原磁场的反作用（弱耦合）。

\- 趋肤深度远大于 $d$，磁场完全穿透。

\- 准静态近似：频率不高，法拉第定律仍适用。



\## 3. 坐标系与磁场表达式



采用柱坐标系 $(r,\\theta,z)$，铝盘位于 $z=0$ 平面。驱动磁铁产生的磁通密度 $z$ 分量为：

$B\_z(r,\\theta,t) = B(r) e^{i(\\Omega\_M t - n\\theta)}$，其中 $n=2$（四极磁铁），复数形式便于计算，物理场取实部。

当磁铁以角速度 $\\Omega\_M$ 旋转时，磁场图案也以相同角速度旋转。在固定坐标系 $(r,\\theta)$ 中，原来在 $\\theta$ 处的磁场值现在出现在 $\\theta - \\Omega\_M t$ 处（因为图案整体转动）。因此：

$B\_z(r,\\theta,t) \\propto \\cos\\left\[2(\\theta - \\Omega\_M t)\\right] = \\cos(2\\theta - 2\\Omega\_M t)$。

利用欧拉公式，余弦可写为复指数函数的实部：

$\\cos(2\\theta - 2\\Omega\_M t) = \\operatorname{Re}\\left\[ e^{i(2\\theta - 2\\Omega\_M t)} \\right] = \\operatorname{Re}\\left\[ e^{-i2\\Omega\_M t} e^{i2\\theta} \\right]$。

但论文中写的是 $e^{i(\\Omega\_M t - n\\theta)}$，即 $e^{i(\\Omega\_M t - 2\\theta)}$。两者相差一个整体负号（共轭）和常数相位，这无关紧要，因为最后要取实部。实际上：

$\\cos(\\Omega\_M t - n\\theta) = \\operatorname{Re}\[e^{i(\\Omega\_M t - n\\theta)}]$，而 $\\cos(n\\theta - n\\Omega\_M t) = \\operatorname{Re}\[e^{i n(\\theta - \\Omega\_M t)}] = \\operatorname{Re}\[e^{-i n\\Omega\_M t} e^{i n\\theta}]$，两者仅相差一个时间反演（$\\Omega\_M \\to -\\Omega\_M$），由于 $\\Omega\_M$ 的方向可以定义，所以两种写法等价。论文采用了 $e^{i(\\Omega\_M t - n\\theta)}$ 的形式。



\## 4. 感应电场与标势



由法拉第定律：$\\nabla \\times \\mathbf{E} = -\\partial \\mathbf{B}/\\partial t$。对于 $z$ 分量：

$(\\nabla \\times \\mathbf{E})\_z = -\\frac{\\partial B\_z}{\\partial t} = -i\\Omega\_M B(r) e^{i(\\Omega\_M t - n\\theta)}$。



铝盘厚度 $d$ 很小（0.1 mm），上下表面为自由边界，没有电极。在稳态下，垂直方向的电流密度 $J\_z$ 必须为零，否则会在表面堆积电荷（因为 $\\nabla \\cdot \\mathbf{J}=0$ 且表面无外接电流）。由欧姆定律 $\\mathbf{J} = \\sigma \\mathbf{E}$（忽略磁场对电流的直接贡献，即霍尔效应很小），得到 $J\_z = \\sigma E\_z$，因此 $E\_z \\approx 0$。这是最主要的物理依据：在良导体薄盘中，垂直电场会被表面电荷迅速屏蔽，使得净 $E\_z$ 几乎为零。



由于铝盘内无自由电荷（$\\nabla \\cdot \\mathbf{E}=0$），引入标势 $\\Phi(r,\\theta,t)$ 使：

$E\_r = \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta},\\quad E\_\\theta = -\\frac{\\partial \\Phi}{\\partial r},\\quad E\_z=0$。

因为在这个场景下的电场无散但有旋，该形式自动满足 $\\nabla \\cdot \\mathbf{E}=0$。代入法拉第定律的 $z$ 分量，经过运算可得泊松方程：

$\\Delta \\Phi = -\\frac{\\partial B\_z}{\\partial t} = -i\\Omega\_M B(r) e^{i(\\Omega\_M t - n\\theta)}$，其中 $\\Delta = \\frac{\\partial^2}{\\partial r^2} + \\frac{1}{r}\\frac{\\partial}{\\partial r} + \\frac{1}{r^2}\\frac{\\partial^2}{\\partial \\theta^2}$ 为二维拉普拉斯算子。



运算：现在将 $E\_r$ 和 $E\_\\theta$ 用 $\\Phi$ 代入：

$E\_r = \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta},\\quad E\_\\theta = -\\frac{\\partial \\Phi}{\\partial r}$。

计算 $\\frac{\\partial}{\\partial r}(r E\_\\theta) = \\frac{\\partial}{\\partial r}\\left( -r \\frac{\\partial \\Phi}{\\partial r} \\right) = -\\frac{\\partial \\Phi}{\\partial r} - r \\frac{\\partial^2 \\Phi}{\\partial r^2}$。

而 $\\frac{\\partial E\_r}{\\partial \\theta} = \\frac{\\partial}{\\partial \\theta}\\left( \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta} \\right) = \\frac{1}{r}\\frac{\\partial^2 \\Phi}{\\partial \\theta^2}$。

因此：

$(\\nabla \\times \\mathbf{E})\_z = \\frac{1}{r}\\left\[ \\left( -\\frac{\\partial \\Phi}{\\partial r} - r \\frac{\\partial^2 \\Phi}{\\partial r^2} \\right) - \\frac{1}{r}\\frac{\\partial^2 \\Phi}{\\partial \\theta^2} \\right] = -\\frac{1}{r}\\frac{\\partial \\Phi}{\\partial r} - \\frac{\\partial^2 \\Phi}{\\partial r^2} - \\frac{1}{r^2}\\frac{\\partial^2 \\Phi}{\\partial \\theta^2}$。

注意到 $-\\frac{1}{r}\\frac{\\partial \\Phi}{\\partial r} - \\frac{\\partial^2 \\Phi}{\\partial r^2} = -\\frac{1}{r}\\frac{\\partial}{\\partial r}\\left( r \\frac{\\partial \\Phi}{\\partial r} \\right)$。所以：

$(\\nabla \\times \\mathbf{E})\_z = -\\left\[ \\frac{1}{r}\\frac{\\partial}{\\partial r}\\left( r \\frac{\\partial \\Phi}{\\partial r} \\right) + \\frac{1}{r^2}\\frac{\\partial^2 \\Phi}{\\partial \\theta^2} \\right]$。

方括号内的表达式正是二维拉普拉斯算子 $\\Delta \\Phi$ 在柱坐标下的形式（注意：二维拉普拉斯没有 $z$ 项）：

$\\Delta \\Phi = \\frac{1}{r}\\frac{\\partial}{\\partial r}\\left( r \\frac{\\partial \\Phi}{\\partial r} \\right) + \\frac{1}{r^2}\\frac{\\partial^2 \\Phi}{\\partial \\theta^2}$。



\## 5. 求解特解（常数磁场近似）



$\\Phi\_{\\text{exact}} = -\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln\\left(\\frac{r}{R}\\right) e^{-i2\\theta}$。



\### 5.1 计算电场分量 $E\_r$



由定义 $E\_r = \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta}$，得：



$E\_r = \\frac{1}{r} \\cdot \\left(-i2\\right) \\Phi = -\\frac{i2}{r} \\Phi = -\\frac{i2}{r} \\left\[ -\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln\\left(\\frac{r}{R}\\right) e^{-i2\\theta} \\right] = -\\frac{i2}{r} \\cdot \\left( -\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln\\frac{r}{R} e^{-i2\\theta} \\right)$。



计算系数：$(-i2) \\cdot (-i) = (i2 \\cdot i) = i^2 \\cdot 2 = -2$，再乘以 $-\\frac{\\Omega\_M B\_0}{4}$？一步步来：



先算括号内：$-\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln\\frac{r}{R} e^{-i2\\theta}$。乘以 $-\\frac{i2}{r}$：



$E\_r = \\left(-\\frac{i2}{r}\\right) \\cdot \\left(-\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln\\frac{r}{R} e^{-i2\\theta}\\right) = \\frac{(i2)(i\\Omega\_M B\_0)}{4r} r^2 \\ln\\frac{r}{R} e^{-i2\\theta}$。



注意负负得正。$(i2)(i\\Omega\_M B\_0) = i^2 \\cdot 2\\Omega\_M B\_0 = -2\\Omega\_M B\_0$。因此：



$E\_r = \\frac{-2\\Omega\_M B\_0}{4r} r^2 \\ln\\frac{r}{R} e^{-i2\\theta} = -\\frac{\\Omega\_M B\_0}{2} r \\ln\\frac{r}{R} e^{-i2\\theta}$。



所以 $E\_r = -\\frac{\\Omega\_M B\_0}{2} r \\ln\\left(\\frac{r}{R}\\right) e^{-i2\\theta}$。注意这里没有虚数单位 $i$，说明 $E\_r$ 是实函数（在给定时刻 $t=0$ 时，实际上我们取了空间部分，时间因子 $e^{i\\Omega\_M t}$ 会带来虚部，但这里我们只考虑振幅）。



\### 5.2 洛伦兹力密度与扭矩



驱动磁场的 $z$ 分量为 $B\_z = B\_0 e^{-i2\\theta}$（忽略时间因子，因为时间平均后只有共轭乘积贡献）。电流密度 $J\_r = \\sigma E\_r$。洛伦兹力密度的切向分量为 $f\_{L\\theta} = \\sigma E\_r B\_z^\*$（取复共轭是因为时间平均 $\\langle \\text{Re}(J\_r) \\text{Re}(B\_z) \\rangle = \\frac{1}{2} \\text{Re}(J\_r B\_z^\*)$，常数因子 $1/2$ 会吸收进最终系数）。因此：



$f\_{L\\theta} \\propto \\sigma E\_r B\_z^\* = \\sigma \\left( -\\frac{\\Omega\_M B\_0}{2} r \\ln\\frac{r}{R} e^{-i2\\theta} \\right) \\cdot \\left( B\_0 e^{i2\\theta} \\right) = -\\frac{\\sigma \\Omega\_M B\_0^2}{2} r \\ln\\frac{r}{R}$。



角度依赖消失，因为 $e^{-i2\\theta} e^{i2\\theta}=1$。



电流密度 $\\mathbf{J} = \\sigma \\mathbf{E}$。洛伦兹力密度 $\\mathbf{f}\_L = \\mathbf{J} \\times \\mathbf{B}$。由于驱动磁场主要沿 $z$ 方向，$\\mathbf{B} \\approx B\_z \\hat{z}$，则：

$\\mathbf{f}\_L = \\sigma (E\_r \\hat{r} + E\_\\theta \\hat{\\theta}) \\times (B\_z \\hat{z}) = \\sigma B\_z (E\_r \\hat{\\theta} - E\_\\theta \\hat{r})$。

扭矩 $z$ 分量为 $T\_z = \\iiint (\\mathbf{r} \\times \\mathbf{f}\_L)\_z dV = \\iiint r f\_{L\\theta} dV$，其中 $f\_{L\\theta} = \\sigma B\_z E\_r$。





\### 5.3 扭矩积分



扭矩 $T\_z = \\int\_0^{2\\pi} \\int\_0^R r \\cdot f\_{L\\theta} \\cdot r dr d\\theta \\cdot d$（厚度 $d$）。代入 $f\_{L\\theta}$：



$T\_z = \\int\_0^{2\\pi} d\\theta \\int\_0^R r \\cdot \\left( -\\frac{\\sigma \\Omega\_M B\_0^2}{2} r \\ln\\frac{r}{R} \\right) \\cdot r dr \\cdot d$。



注意：$r$（力臂）乘以 $f\_{L\\theta}$ 再乘以面积元 $r dr d\\theta$ 给出 $r^2 dr d\\theta$ 因子？正确写法：$dT\_z = r \\cdot f\_{L\\theta} \\cdot (r dr d\\theta) \\cdot d$，所以被积函数有 $r^2 f\_{L\\theta}$。而 $f\_{L\\theta} \\propto r \\ln(r/R)$，因此总的是 $r^3 \\ln(r/R)$。



所以：



$T\_z = -\\frac{\\sigma \\Omega\_M B\_0^2 d}{2} \\int\_0^{2\\pi} d\\theta \\int\_0^R r^3 \\ln\\frac{r}{R} dr$。



角度积分给出 $2\\pi$。径向积分：



$I = \\int\_0^R r^3 \\ln\\frac{r}{R} dr$。



令 $x = r/R$，则 $r = R x$，$dr = R dx$，$r^3 = R^3 x^3$，$\\ln(r/R) = \\ln x$。于是：



$I = \\int\_0^1 R^3 x^3 \\cdot \\ln x \\cdot R dx = R^4 \\int\_0^1 x^3 \\ln x \\, dx$。



计算 $\\int\_0^1 x^3 \\ln x \\, dx = \\left\[ \\frac{x^4}{4} \\ln x \\right]\_0^1 - \\int\_0^1 \\frac{x^4}{4} \\cdot \\frac{1}{x} dx = 0 - \\frac{1}{4} \\int\_0^1 x^3 dx = -\\frac{1}{4} \\cdot \\frac{1}{4} = -\\frac{1}{16}$。



因此 $I = R^4 \\cdot \\left(-\\frac{1}{16}\\right) = -\\frac{R^4}{16}$。



代入扭矩表达式：



$T\_z = -\\frac{\\sigma \\Omega\_M B\_0^2 d}{2} \\cdot (2\\pi) \\cdot \\left(-\\frac{R^4}{16}\\right) = \\frac{\\sigma \\Omega\_M B\_0^2 d \\cdot 2\\pi \\cdot R^4}{32} = \\frac{\\pi}{16} \\sigma d B\_0^2 R^4 \\Omega\_M$。





\## 6. 考虑铝盘旋转（相对速度）



若铝盘以角速度 $\\Omega\_D$ 旋转，则铝盘感受到的磁场旋转角速度为 $\\Omega\_M - \\Omega\_D$。因此只需将上述表达式中的 $\\Omega\_M$ 替换为 $\\Omega\_M - \\Omega\_D$：

$E\_\\theta \\approx -\\frac{i(\\Omega\_M - \\Omega\_D) B(r)}{2} r e^{i\[(\\Omega\_M - \\Omega\_D) t - n\\theta]}$。

这是驱动扭矩来源于\*\*相对转速差\*\*的关键。







虽然 $\\Phi$ 本身误差很大，但最终扭矩 $T\_z$ 是通过对 $\\Phi$ 的导数（电场）进行面积分得到的。我们计算一下两种情况下 $T\_z$ 的表达式：



近似解（论文推导）：



$E\_r = \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta} = -i2 \\Phi / r \\propto -i2 \\cdot \\frac{i\\Omega\_M B\_0}{4} r e^{-i2\\theta} = \\frac{\\Omega\_M B\_0}{2} r e^{-i2\\theta}$



$J\_r = \\sigma E\_r$，洛伦兹力 $f\_{L\\theta} = \\sigma E\_r B\_z$（取时间平均后），积分得 $T\_z \\propto \\sigma d B\_0^2 R^4 \\Omega\_M$（正比于 $R^4$）。



精确解（满足边界条件 $\\Phi(R,\\theta)=0$）：



$\\Phi = -\\frac{i\\Omega\_M B\_0}{4} r^2 \\ln(r/R) e^{-i2\\theta}$



$E\_r = \\frac{1}{r}\\frac{\\partial \\Phi}{\\partial \\theta} = -i2 \\Phi / r = -i2 \\cdot \\left(-\\frac{i\\Omega\_M B\_0}{4}\\right) r \\ln(r/R) e^{-i2\\theta} = -\\frac{\\Omega\_M B\_0}{2} r \\ln(r/R) e^{-i2\\theta}$



注意：$\\ln(r/R)$ 在 $r<R$ 时为负，所以 $E\_r$ 的符号与近似解相同（因为负负得正？近似解 $E\_r \\propto +\\frac{\\Omega\_M B\_0}{2} r$，精确解 $E\_r \\propto -\\frac{\\Omega\_M B\_0}{2} r \\ln(r/R)$，由于 $\\ln(r/R)<0$，所以 $-\\ln(r/R)>0$，两者同号）。



扭矩表达式（忽略常数因子）为：

$T\_z \\propto \\int\_0^R r \\cdot f\_{L\\theta} \\cdot r dr = \\int\_0^R r^2 f\_{L\\theta} dr$（因为 $dV = r dr d\\theta dz$，且已对 $\\theta$ 和 $z$ 积分）。



而 $f\_{L\\theta} = \\sigma E\_r B\_z$，其中 $E\_r \\propto \\frac{\\Omega\_M B\_0}{2} r \\ln(r/R)$（精确解），$B\_z \\propto B\_0$，所以 $f\_{L\\theta} \\propto \\sigma B\_0^2 \\Omega\_M r \\ln(r/R)$。



于是：

$T\_z \\propto \\int\_0^R r^2 \\cdot \\left( r \\ln(r/R) \\right) dr = \\int\_0^R r^3 \\ln(r/R) dr$。



计算该积分：

令 $I = \\int\_0^R r^3 \\ln(r/R) dr$。做变量代换 $x = r/R$，则 $r = Rx$，$dr = R dx$：

$I = \\int\_0^1 (R x)^3 \\ln(x) \\cdot R dx = R^4 \\int\_0^1 x^3 \\ln x , dx$。



已知 $\\int\_0^1 x^m \\ln x , dx = -\\frac{1}{(m+1)^2}$（$m > -1$）。这里 $m=3$，所以 $\\int\_0^1 x^3 \\ln x , dx = -\\frac{1}{(4)^2} = -\\frac{1}{16}$。



因此 $I = R^4 \\cdot \\left(-\\frac{1}{16}\\right) = -\\frac{R^4}{16}$。



所以精确扭矩 $T\_z^{\\text{exact}} \\propto \\sigma d B\_0^2 \\Omega\_M R^4$，与近似解的形式完全一致（都是 $R^4$），只是常数因子不同（精确解常数包含 $1/16$，而近似解可能为 $1/4$ 或 $1/2$）。我之前错误地用了 $r^2$ 积分，得出 $R^3$，现已更正。



关键结论：尽管 $\\Phi$ 的分布差异很大，但扭矩积分后仍然得到 $T\_z \\propto \\sigma d B\_0^2 R^4 \\Omega\_M$ 的比例关系，只是比例常数不同。这个常数在论文中被吸收进 $\\alpha$ 和 $\\beta$ 系数中，并通过实验标定（使用已知粘度的参考气体）确定。



\## 粘性扭矩 $T\_v$ 的严密推导（对应原文公式 (5)）



\### 1. 物理模型



\- 探针盘（半径为 $R$，厚度可忽略）悬浮在底部磁铁上方，两者之间的间隙为 $h$（$h \\ll R$）。

\- 间隙内充满待测气体，气体为牛顿流体，粘度系数 $\\eta$。

\- 探针盘以角速度 $\\Omega\_D$ 绕中心轴旋转，底部磁铁上表面静止。

\- 由于 $h \\ll R$，间隙内的流动可近似为两平行平板间的库埃特流动（Couette flow），忽略曲率效应。



\### 2. 速度分布



在柱坐标系 $(r,\\theta,z)$ 中，$z=0$ 为底部磁铁上表面，$z=h$ 为探针盘下表面。假设：

\- 流动仅为周向流动，$v\_r = v\_z = 0$，$v\_\\theta = v\_\\theta(r,z)$。

\- 由于 $h$ 很小，在局部范围内（固定 $r$），速度沿 $z$ 方向线性分布（牛顿流体在简单剪切下的稳态解）：



$v\_\\theta(r,z) = r \\Omega\_D \\cdot \\frac{z}{h}, \\quad 0 \\le z \\le h$。



其中 $r\\Omega\_D$ 是探针盘下表面在半径 $r$ 处的线速度，底部速度为零。该线性分布满足无滑移边界条件。



\### 3. 切应力



对于牛顿流体，切应力分量 $\\tau\_{z\\theta}$ 与速度梯度成正比：



$\\tau\_{z\\theta} = \\eta \\frac{\\partial v\_\\theta}{\\partial z}$。



代入速度分布：



$\\frac{\\partial v\_\\theta}{\\partial z} = r \\Omega\_D \\cdot \\frac{1}{h}$，



因此：



$\\tau\_{z\\theta} = \\eta \\frac{r \\Omega\_D}{h}$。



该应力作用在探针盘下表面（$z=h$），方向与 $v\_\\theta$ 相反（气体对盘施加阻力）。



\### 4. 粘性扭矩计算



在探针盘下表面取面积元 $dA = r dr d\\theta$。该面积元上受到的切向力为 $dF = \\tau\_{z\\theta} \\, dA$，方向沿 $\\theta$ 负向。该力对中心轴的扭矩（$z$ 分量）为：



$dT\_v = -r \\cdot dF$（负号表示阻力矩）。



于是：



$dT\_v = -r \\cdot \\left( \\eta \\frac{r \\Omega\_D}{h} \\right) \\cdot r dr d\\theta = -\\eta \\frac{\\Omega\_D}{h} r^3 dr d\\theta$。



对整个圆盘面积分：



$T\_v = \\int\_0^{2\\pi} \\int\_0^R -\\eta \\frac{\\Omega\_D}{h} r^3 dr d\\theta = -\\eta \\frac{\\Omega\_D}{h} \\int\_0^{2\\pi} d\\theta \\int\_0^R r^3 dr$。



计算积分：



$\\int\_0^{2\\pi} d\\theta = 2\\pi$，



$\\int\_0^R r^3 dr = \\frac{R^4}{4}$。



因此：



$T\_v = -\\eta \\frac{\\Omega\_D}{h} \\cdot 2\\pi \\cdot \\frac{R^4}{4} = -\\frac{\\pi}{2} \\frac{\\eta R^4 \\Omega\_D}{h}$。



\### 5. 与原文公式的比较



原文公式 (5) 写为：



$(T\_v)\_z \\approx -\\frac{\\eta R^4 \\Omega\_D}{h}$。



两者相差常数因子 $\\pi/2$。原文中使用了近似等号 $\\approx$，并将该常数因子吸收进后续的系数 $\\alpha$ 中（见公式 (9) 中的 $C\_1$）。因此，在扭矩平衡方程中，实际使用的是比例关系 $T\_v \\propto -\\eta R^4 \\Omega\_D / h$，具体常数通过实验标定确定。



\### 6. 讨论与近似条件



\- \*\*线性速度分布的有效性\*\*：当 $h \\ll R$ 且雷诺数 $Re = \\rho \\Omega\_D R h / \\eta \\ll 1$ 时，惯性项可忽略，斯托克斯方程的解确实为线性分布。对于气体，$\\eta$ 很小，但 $\\rho$ 也很小，通常满足层流条件。

\- \*\*曲率修正\*\*：严格解应考虑柱坐标下的纳维-斯托克斯方程，得到的速度分布包含 $r$ 的修正项，但量级为 $O(h/R)$，在 $h \\ll R$ 时忽略。

\- \*\*边缘效应\*\*：在 $r \\approx R$ 处，流动不再是纯剪切，存在径向压力梯度。但边缘贡献与面积相比为 $O(h/R)$，同样可忽略。



因此，公式 $T\_v \\approx -\\eta R^4 \\Omega\_D / h$ 抓住了主导项，且通过实验标定可以补偿常数因子，是合理的近似。



\## 基座磁铁完全轴对称时电磁制动扭矩为零的推导



在均匀圆柱体产生的势场中，势能只依赖于柱坐标下的径向坐标 $\\rho$（即到圆柱轴线的距离），即 $V = V(\\rho,z)$。拉格朗日函数在柱坐标 $(\\rho, \\phi, z)$ 下为：

$$L = \\frac{1}{2}m(\\dot{\\rho}^2 + \\rho^2\\dot{\\phi}^2 + \\dot{z}^2) - V(\\rho, z).$$



\### 守恒量

\- \*\*角动量的 $z$ 分量\*\*守恒：$L\_z = m\\rho^2\\dot{\\phi}$ 为常数。



\### 计算 $\\frac{d}{dt}(\\mathbf{r} \\times \\mathbf{p})$

在笛卡尔坐标下，$\\mathbf{r} \\times \\mathbf{p}$ 的导数满足：

$$\\frac{d}{dt}(\\mathbf{r} \\times \\mathbf{p}) = \\mathbf{r} \\times \\dot{\\mathbf{p}} + \\dot{\\mathbf{r}} \\times \\mathbf{p} = \\mathbf{r} \\times \\mathbf{F} + \\mathbf{v} \\times (m\\mathbf{v}) = \\mathbf{r} \\times \\mathbf{F},$$

其中 $\\mathbf{F} = -\\nabla V$。在柱坐标中，$\\mathbf{r} = \\rho\\,\\mathbf{e}\_\\rho + z\\,\\mathbf{e}\_z$，$\\mathbf{F} = F\_\\rho\\,\\mathbf{e}\_\\rho = -\\frac{dV}{d\\rho}\\,\\mathbf{e}\_\\rho$，因此：

$$\\mathbf{r} \\times \\mathbf{F} = (\\rho\\,\\mathbf{e}\_\\rho + z\\,\\mathbf{e}\_z) \\times (F\_\\rho\\,\\mathbf{e}\_\\rho) = zF\_\\rho\\,(\\mathbf{e}\_z \\times \\mathbf{e}\_\\rho) = zF\_\\rho\\,\\mathbf{e}\_\\phi.$$

故 $\\frac{d}{dt}(\\mathbf{r} \\times \\mathbf{p}) = z\\left(-\\frac{dV}{d\\rho}\\right)\\mathbf{e}\_\\phi$。该结果表明角动量的 $z$ 分量守恒（因 $\\mathbf{e}\_\\phi$ 无 $z$ 分量），而其他分量不守恒。



$L\_z$ 守恒代表绕 $z$ 轴的转动（即角速度方向沿 $z$ 轴）不会发生变化。
