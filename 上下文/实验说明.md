# 抗磁悬浮电磁旋转粘度计

## 实验简介

本装置利用高定向热解石墨（HOPG）的抗磁性实现探针的无接触悬浮，通过旋转磁场驱动探针在待测气体中旋转，利用视频转速跟踪技术测量探针转速，基于转速比与粘度的线性关系实现气体粘度的精密测量。

## 实验原理

### 1. 抗磁悬浮

HOPG 具有强抗磁性（磁化率 $\chi \approx -4 \times 10^{-4}$），在 NdFeB 永磁体产生的非均匀磁场中受到指向磁场极小值处的恢复力，可实现稳定悬浮。这一机制由 Earnshaw 定理的逆应用保证，从根源上消除了机械摩擦。

### 2. 涡流驱动

顶部驱动磁铁（四极结构）以角速度 $\Omega_M$ 旋转，产生旋转磁场。磁场穿过紧贴石墨盘的铝质圆盘，在其中感应涡流。载流铝盘在磁场中受到洛伦兹力，产生驱动扭矩：

$$T_z \propto \sigma d B^2 R^4 (\Omega_M - \Omega_D)$$

其中 $\sigma$ 为铝盘电导率，$d$ 为厚度，$B$ 为磁场幅值，$R$ 为铝盘半径，$\Omega_D$ 为探针旋转角速度。

### 3. 粘性阻力矩

探针盘在气体中旋转时，底部间隙（高度 $h \ll R$）内形成 Couette 剪切流动。根据牛顿粘性定律：

$$T_v \approx -\frac{\eta R^4 \Omega_D}{h}$$

其中 $\eta$ 为气体粘度，负号表示阻力矩。

### 4. 稳态力矩平衡

稳态旋转时，驱动力矩与粘性阻力矩平衡（电磁制动扭矩在轴对称磁场中为零）：

$$T_z + T_v = 0$$

代入扭矩表达式，整理得：

$$\eta = \alpha \cdot \frac{\Omega_M}{\Omega_D} + \beta$$

即**粘度 $\eta$ 与转速比 $\Omega_M/\Omega_D$ 成线性关系**。系数 $\alpha$、$\beta$ 通过测量已知粘度的标准气体标定。

## 装置结构

| 组件 | 说明 |
|------|------|
| 基座磁铁 | 两个径向充磁 NdFeB 磁铁嵌套，产生轴对称磁场 |
| HOPG 石墨圆盘 | 强抗磁性材料，在基座磁铁上方稳定悬浮 |
| 铝质探针盘 | 紧贴石墨盘，感应涡流产生驱动力矩 |
| 驱动磁铁 | 四块方形磁铁，步进电机驱动旋转 |
| 密封腔 | 亚克力桶，可抽真空/注入待测气体 |
| 黑线标记 | 探针表面的黑色标记线，用于转速检测 |

## 实验步骤

1. **样品准备**：将待测气体注入密封腔，重复换气三次确保纯度
2. **悬浮调试**：将 HOPG 探针置于基座磁铁上方，使其稳定悬浮
3. **旋转驱动**：启动步进电机，驱动磁铁以设定转速 $\Omega_M$ 旋转
4. **视频记录**：用手机/相机录制探针旋转过程（建议 60fps 以上）
5. **数据分析**：上传视频 → 框选探针区域 → 转速跟踪 → 数据拟合 → 计算粘度

## 转速跟踪方法

使用 OpenCV 进行视频逐帧分析：

- **ROI 框选**：在第一帧上用矩形框选探针附近区域，自动转椭圆参数
- **极坐标映射**：将椭圆区域展开为极坐标图像
- **黑线检测**：沿径向方向求平均后高斯平滑，用抛物插值实现亚像素角度定位
- **RPM 计算**：帧间瞬时转速 + 一圈滑动平均 + 过零检测（三种方法互补）
- **低置信度处理**：当黑线模糊时，用历史角速度预测当前角度

## 数据拟合

实验数据预期满足 $\eta \propto \Omega_M/\Omega_D$ 的线性关系。本平台支持 6 种拟合模型：

| 模型 | 公式 | 适用场景 |
|------|------|----------|
| 线性 | $y = ax + b$ | 标准粘度-转速比关系 |
| 正比例 | $y = kx$ | 截距为零的简化模型 |
| 二次 | $y = ax^2 + bx + c$ | 存在非线性修正时 |
| 指数 | $y = a e^{bx}$ | 特殊流动状态 |
| 对数 | $y = a \ln x + b$ | 特定流变行为 |
| 反比例 | $y = k/x + b$ | 辅助分析 |

## 技术优势

1. **零摩擦测量**：完全非接触悬浮与驱动，消除机械摩擦系统误差
2. **高灵敏度**：可测量 $\mu$Pa·s 量级气体粘度，较传统方法提升约两个数量级
3. **宽量程**：通过调节磁铁间距动态优化磁场耦合效率，适应不同粘度范围
4. **洁净测量**：密封腔设计，适用于有毒、高纯气体
